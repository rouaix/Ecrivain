Accès IA — MCP

Connectez une IA à vos projets via le protocole MCP

Le Model Context Protocol (MCP) permet à une IA comme Claude Desktop d'interagir directement avec vos projets : lire vos chapitres, créer du contenu, rechercher dans votre texte… depuis une conversation IA, sans passer par le navigateur.

1 Cliquez sur Générer un token ci-dessous.
2 Copiez la configuration qui apparaît et collez-la dans les paramètres de Claude Desktop.

Un token donne un accès complet à vos projets. Ne le partagez pas. Révoquez-le immédiatement s'il est compromis.
